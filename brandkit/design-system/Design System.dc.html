Dark rules: no shadows — borders only. Signal #3BD1B5 becomes the accent; solid Echo Teal is reserved for light surfaces.

Behavior & responsive rules

  • States: every interactive component defines default / hover / focus-visible (2px teal ring) / disabled (50% opacity) / loading (spinner + aria-busy).
  • Spacing: 4px base scale; space-y-8 between sections, space-y-4 within cards; targets ≥ 40×40px.
  • Responsive: single column under 768px; tables scroll horizontally inside their card (overflow-x-auto) — the page body never scrolls sideways; the URL form stacks input over button on mobile.
  • Accessibility: real button/label/table semantics; never color-only status; prefers-reduced-motion disables the pulse and gauge sweep.
