## 1. Session summary

**Completed:** the entire MVP, Phases 0–3 of
[implementation-plan.md](../implementation-plan.md), verified end-to-end.

- **Phase 0:** `git init` + baseline; `.gitignore`; README fixes (doc links,
  Node 20+); `CONTRIBUTING.md`, `SECURITY.md`.
- **Docs:** all 7 empty placeholders authored (architecture, design + 16 ADRs,
  roadmap, test-suite, session-rules, agent-workflows, implementation-plan),
  then reconciled claim-by-claim against the implemented code.
- **Backend:** FastAPI api + worker (one image), SQLAlchemy 2.0 + Alembic,
  Postgres-as-queue (`FOR UPDATE SKIP LOCKED`, stale-claim reaper, attempts>3
  → failed), 6-step pipeline (discovery → KYC → prompts → execute → footprint
  → scoring), Provider interface (Claude + OpenAI real, Gemini + Perplexity
  stubbed, deterministic MockProvider under `DRY_RUN=1`), `llm_cache` (24h),
  SSRF guard (`app/net_guard.py`).
- **Frontend:** Next.js 15, 3 screens per brandkit (UrlForm → StepProgress →
  ScoreGauge/ResultsTable/KYC/prompts + failure card), generated types +
  hand-maintained `lib/contracts.ts` alias layer.
- **Infra:** Makefile control panel, dev/prod compose (dev host ports
  parameterized), deploy/rollback scripts (untested), CI workflows (unproven —
  no remote), OpenAPI gen-types contract (byte-stable).
- **Verification:** `make lint`/`typecheck`/`test` green — **62 backend tests**
  (incl. real-Postgres SKIP LOCKED tests) **+ 9 vitest**; live DRY_RUN stack
  driven end-to-end twice (pre- and post-review): score 0.6, 40 responses, 24
  footprints; failure, 422, and SSRF-probe paths verified; web proxy +
  landing page verified. Playwright spec written but env-blocked (see debt).
- **Adversarial review:** 5 dimensions → 23 findings → 16 confirmed by
  independent skeptics → **all 16 fixed** (7 refuted with reasons on record).

**Why this way:** the repo had an aspirational README/PRD and zero code; the
fastest honest path was to lock a SPEC, build to the docs, then force docs and
code to converge by verification rather than trust.
